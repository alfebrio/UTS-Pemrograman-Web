Badminton

Saya telah bermain Badminton sejak Kelas 2 SD.

Selama masa SD - SMP, Saya telah mengikuti berbagai Kejurkab dan Kejurnas, bahkan Kejuaraan Tarkam hingga Saya mengalami cedera di bagian pergelangan tangan. Sejak itu Saya berhenti bermain hingga bertahun-tahun.

Untuk saat ini saya sering menyempatkan untuk bermain kurang lebih seminggu 2x sebagai Hobi, dan terkadang mengikuti Kejuaraan kecil sebagai kegiatan.

Gaming

Untuk Hobi ini sendiri Saya telah disuguhi PlayStation sejak berumur 3,5 tahun.

Game yang pertama kali saya mainkan adalah Seri Digimon dan Rumble Racing. Sejak saat itu Saya tidak pernah bosan bermain berbagai genre game, terutama Arcade dan Racing.

Saat SMP, Anak-anak mulai bermain di platform Mobile, dan Saya pun mengikuti alur. Game mobile online pertama yang saya mainkan adalah Clash of Clans.

Suatu waktu terdapat Game moba baru yang dirilis bernama Mobile Legends. Saya cukup excited karena sebelumnya Saya cukup sering bermain Game moba PC yaitu Dota 2. Sejak saat itu saya sedikit aktif di Game tersebut hingga dapat memenangkan beberapa kompetisi lokal dan menjadi penjoki untuk menambah uang saku. Saya juga bermain berbagai Game lain di platform PC

Jadi, kurang lebih Saya adalah pemain di semua platform Game
